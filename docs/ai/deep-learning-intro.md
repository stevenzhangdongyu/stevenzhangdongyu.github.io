# 深度学习入门：神经网络基础

## 什么是深度学习？

深度学习是机器学习的一个分支，它基于人工神经网络，特别是深度神经网络来解决复杂的问题。与传统机器学习不同，深度学习能够自动从数据中提取特征，而不需要人工进行特征工程。

## 神经网络的基本结构

### 感知机

感知机是最简单的神经网络单元，它接收多个输入，通过加权求和后经过激活函数产生输出：

```
y = f(Σ(wi * xi) + b)
```

其中：
- `xi` 是输入
- `wi` 是权重
- `b` 是偏置
- `f` 是激活函数

### 多层感知机 (MLP)

多层感知机由输入层、隐藏层和输出层组成：

- **输入层**：接收原始数据
- **隐藏层**：进行特征变换和提取
- **输出层**：产生最终预测结果

## 常用激活函数

| 激活函数 | 公式 | 特点 |
|---------|------|------|
| ReLU | max(0, x) | 计算简单，缓解梯度消失 |
| Sigmoid | 1/(1+e^(-x)) | 输出范围(0,1)，适合二分类 |
| Tanh | (e^x - e^(-x))/(e^x + e^(-x)) | 输出范围(-1,1)，零中心 |
| Softmax | e^xi/Σe^xj | 多分类输出 |

## 训练过程

神经网络的训练主要包括以下步骤：

1. **前向传播**：数据通过网络得到预测结果
2. **计算损失**：比较预测值与真实值的差异
3. **反向传播**：计算梯度，更新权重
4. **迭代优化**：重复上述过程直到收敛

## 实践建议

- 从简单的模型开始，逐步增加复杂度
- 关注数据质量，好的数据比复杂模型更重要
- 使用正则化技术防止过拟合
- 合理设置学习率和批量大小
